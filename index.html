3. What are semantic tags?


Semantics tags in HTML are special markup tags that are used to describe the meaning and
structure of the content within a web page. These tags are designed to provide more
context and information to both web browsers and search engines about the purpose of the
content, which can help improve accessibility, search engine optimization, and the overall
user experience.
Some examples of semantic tags in HTML5 include:

 <header> : Used to define the header of a page or section.
<nav> : Used to define a navigation menu.
<article> : Used to define an article or section of content within a page.
<section> : Used to define a section of content within a page.
<aside> : Used to define content that is tangentially related to the main content,
such as a sidebar or callout box.
<footer> : Used to define the footer of a page or section.

4. Write code to create an ordered list with roman numbers.


Ordered list in html is used to create list elements which are following some
specific order. Its default type is number or "1". Other types which it supports
are "I" , "i", "A", "a"

Code:-


<ol type = "I" >
<li> Apple </li>
<li> Orange </li>
<li> Banana </li>
</ol >

Output:-


  1. Apple
  2. Orange
  3. Banana

5. Write code to create an unordered list with a circle.


Unordered list in html is used to create list elements which are random in
fashion. Its default type is bullet or "disc". Other types which it supports are
"circle" , "square", "none".


Code:-



<ul type = "circle">
<li> First </li>
<li> Second </li>
<li> Third </li>
</ul>

Output:-


  • First
  • Second
  • Third
